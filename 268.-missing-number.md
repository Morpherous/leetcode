# 268. Missing Number

```java
class Solution {
    public int missingNumber(int[] nums) {
        int res=0;
        for(int i=0;i<nums.length;i++){
            res^=i;
            res^=nums[i];
        }
        res^=nums.length;
        return res;
    }
}
```

53-3：数组中数值和下标相等的元素

![](<.gitbook/assets/image (2) (1).png>)

```java
public int missingNumber(int[] nums) {
       int L=0,R=nums.length-1;
       while(L<=R){
           int mid=L+(R-L)/2;
           if(nums[mid]==mid){
               return mid;
           }
           if(nums[mid]>mid){
               R=mid-1;
           }else {
               L=mid+1;
           }
       }
       if(L==nums.length){
           return L;
       }
       return -1;
    }
```
